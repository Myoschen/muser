## Project Setup

### 1. Clone the repository

```bash
git clone https://github.com/Myoschen/muser.git
```

### 2. Install dependencies

```bash
pnpm install
```

### 3. Run in Development mode

```bash
pnpm run dev
```

### 4. Build the project

```bash
# For windows
$ pnpm run build:win

# For macOS
$ pnpm run build:mac

# For Linux
$ pnpm run build:linux
```
